**1:Local-First**
User's data get stored in users device itself.

**2:Project Centric**
Everything belongs to a project, not a chat.

**3:Structured Memory**
- We are trying to store knowledge not just converstation.
- It wont work like all the chat is saved as a memory, but the important knowledge is saved  in the memory so that even if we delete that thread the important point should be still saved in the user memory.

**4:Modular**
Every component should be replaceable:

AI provider
Database (if needed)
Memory engine
UI

**5:AI Assists, Not just Chats**:
If the AI can perform or organize a task instead of just answering, that's usually the better experience.
